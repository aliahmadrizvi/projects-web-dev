🎬 NoPoint Media

A simple HTML5 media website that showcases different categories of YouTube content, including Daily Vlogs, Editz, Live IRL Streams, and Comedy Videos. The website uses embedded YouTube videos for an easy viewing experience and provides simple navigation between sections.

---

📌 Features

- 🎥 Daily Vlogs section
- ✂️ Editz collection
- 🔴 Live IRL Streams
- 😂 Comedy Videos
- 📺 Embedded YouTube videos using "iframe"
- 🧭 Navigation menu with internal links
- 📝 About section
- 🔝 "Go Back to Top" navigation
- 📱 Responsive viewport support

---

🛠️ Technologies Used

- HTML5
- YouTube Embed ("iframe")

🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open index.html in your preferred web browser.
4. Browse different video categories using the navigation menu.
