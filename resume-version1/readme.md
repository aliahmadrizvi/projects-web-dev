📄 Resume Website v1.0

A simple HTML5 Resume Website showcasing my profile, education, skills, projects, achievements, and contact details. This was one of my first web development projects built using only HTML.

✨ Features

- 👤 Personal profile
- 🎓 Education
- 💼 Work Experience section
- 💻 Skills
- 🏆 Achievements
- 📂 Projects with GitHub links
- ❓ FAQ section using "<details>" and "<summary>"
- 📞 Contact information

🛠️ Tech Stack

- HTML5
