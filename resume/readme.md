

📄 Resume Website

- 👤 Professional profile
- 🎓 Education details
- 💻 Skills section
- 📂 Projects
- 🏆 Achievements & Certifications
- ❤️ Hobbies
- 📞 Contact information
- 🎨 Modern UI with gradient background

🛠️ Tech Stack

- HTML5
- CSS3

Resume-Website/
