
💼 Personal Portfolio Website

A modern and responsive Personal Portfolio Website built using HTML5 and CSS3. This project showcases my profile, skills, projects, and contact information while serving as my personal portfolio for learning web development.

---

🌟 Features

- 🏠 Hero/Home section
- 👤 About Me section
- 💻 Skills section
- 📂 Projects section
- 📞 Contact section
- 🧭 Sticky navigation bar
- 🎨 Modern UI with gradient background
- 🖼️ Circular profile picture
- 🔗 GitHub profile link
- 📱 Smooth scrolling navigation
- ✨ Hover effects on navigation links

---

🛠️ Technologies Used

- HTML5
- CSS3

---

---

🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Ensure the "images" folder contains "profilepic.jpg".
4. Open "index.html" in your preferred web browser.
